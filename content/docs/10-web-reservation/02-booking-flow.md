# 予約フロー

## できること
- 予約の変更・キャンセルができる
- 予約履歴を確認できる
- リピート予約を簡単に行える

## 事前準備
- 予約済みの予約番号またはログイン済みアカウントがあること

## 手順
1. マイページにログイン
2. 「予約履歴」をクリック
3. 変更・キャンセルしたい予約を選択
4. 変更の場合は新しい日時を選択
5. キャンセルの場合は「キャンセル」をクリック
6. 確認ダイアログで「OK」を押す

## 確認ポイント
- 変更後の日時が正しく反映されていること
- キャンセル完了メールが届くこと
- キャンセル料が発生する場合は金額が表示されること

## よくあるミス
- キャンセル期限を過ぎた予約をキャンセルしようとする
- 変更先の時間枠が既に埋まっている
