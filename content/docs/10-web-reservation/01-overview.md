# Web予約

## できること
- お客様がWebブラウザから指名予約・フリー予約を行える
- 空き枠をリアルタイムで確認できる
- 予約完了後に確認メールが自動送信される

## 事前準備
- 店舗情報の登録が完了していること
- キャスト情報・シフトが登録されていること
- メール通知設定が有効になっていること

## 手順
1. トップページの「予約する」ボタンをクリック
2. 希望日時を選択
3. コース・オプションを選択
4. 指名キャストを選択（フリーの場合はスキップ）
5. お客様情報を入力
6. 予約内容を確認し「予約確定」をクリック

## 確認ポイント
- 予約完了画面に予約番号が表示されること
- 確認メールが届くこと
- 管理画面の予約一覧に反映されること

## よくあるミス
- シフト未登録のキャストを指名しようとしてエラーになる
- メールアドレスの入力ミスで確認メールが届かない
