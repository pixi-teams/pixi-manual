# システム設定

## できること
- 店舗の基本情報を設定できる
- 営業時間・定休日を設定できる
- 通知メールのテンプレートを編集できる
- システム全体の動作設定を変更できる

## 事前準備
- 管理者アカウントでログインしていること

## 手順
1. サイドメニューから「設定」を選択
2. 「店舗情報」タブで基本情報を入力
3. 「営業時間」タブで営業時間と定休日を設定
4. 「通知設定」タブでメールテンプレートを編集
5. 「保存」ボタンをクリック

## 確認ポイント
- 設定変更がWeb予約画面に正しく反映されること
- テスト通知メールが正しく送信されること

## よくあるミス
- 営業時間の開始と終了が逆になっている
- メールテンプレートの変数名を間違えて入力する
