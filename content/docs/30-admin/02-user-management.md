# ユーザー管理

## できること
- 管理画面のユーザーアカウントを作成・編集・削除できる
- ロール（権限）を割り当てられる
- パスワードのリセットができる

## 事前準備
- 管理者アカウントでログインしていること

## 手順
1. サイドメニューから「ユーザー管理」を選択
2. 「新規追加」ボタンをクリック
3. ユーザー名・メールアドレス・ロールを入力
4. 「作成」ボタンをクリック
5. 初期パスワードが記載されたメールが送信される

## 確認ポイント
- 新規ユーザーが一覧に表示されること
- 割り当てたロールに応じた画面のみ表示されること
- 初期パスワードでログインできること

## よくあるミス
- 既に使用されているメールアドレスを登録しようとする
- ロールの設定を間違えて権限が不足する
